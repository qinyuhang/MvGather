MvGather
========

<<<<<<< HEAD
MvGather中文名影视集结号，是个在线视频客户端，同时也是本地视频播放器。
基于Qt5.3及QtAV开发，同时使用QCFramer皮肤框架。
主要用于解决ubuntu及其它linux发行版上网页观看视频占用大量CPU。
